
# Nucleares-AchievementCheat
Automatically unlock all achievements in the game "Nucleares" using BepInEx and Harmony.

 1. Download the Mod.
Mosey on over to the 'releases' section and download the latest Nucleares-AchievementCheat.dll -- that's all you need.
https://github.com/kougyokugentou/Nucleares-AchievementCheat/releases
 2. Install BepInEx (Prerequisite)
 a.  Go to the official BepInEx releases page: [https://github.com/BepInEx/BepInEx/releases](https://github.com/BepInEx/BepInEx/releases)     
b.  Download the appropriate version (usually **BepInEx_x64_5.x.x.x.zip**).  
c.  Extract all files into your game's root directory (the folder containing the .exe).  
d.  Run the game once. Wait until the main menu loads, then exit.  
e.  This will automatically generate the **BepInEx/plugins/** and **BepInEx/config/** folders.

 3. Install the Mod.
Place the downloaded **Nucleares-AchievementCheat.dll** into:  
`Game Directory/BepInEx/plugins/`

4. Run the game and load into a new game (or load an existing one).
5. Achievements should proc. If not, fiddle with the reactor for a bit such as insert your fuel cell into the core.

**Problems?**
Check the BepInEx logs and report issues on GitHub.
